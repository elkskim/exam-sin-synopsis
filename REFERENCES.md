## HOW TO CITE IN LaTeX

Add to your main.tex before \end{document}:

```latex
\begin{thebibliography}{99}

\bibitem{fowler2006}
Fowler, M., \& Foemmel, M. (2006). 
\textit{Continuous Integration}. 
Retrieved from https://martinfowler.com/articles/continuousIntegration.html

\bibitem{newman2021}
Newman, S. (2021). 
\textit{Building Microservices: Designing Fine-Grained Systems} (2nd ed.). 
O'Reilly Media.

\bibitem{github2024}
GitHub. (2024). 
\textit{GitHub Actions Documentation}. 
Retrieved from https://docs.github.com/en/actions

% Add more as needed...

\end{thebibliography}
```

Then reference in text like: "According to Fowler \cite{fowler2006}, continuous integration..."


## QUICK REFERENCE TIPS

- Aim for 5-10 quality references minimum
- Mix of: academic papers, industry books, official documentation
- Don't forget to cite course materials!
- Use specific sections/chapters when possible
- Include URLs with "Retrieved from" and date if online sources
